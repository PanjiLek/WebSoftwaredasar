VLC Media Player

Apa Itu VLC Media Player?

VLC Media Player adalah pemutar media gratis dan open-source yang dikembangkan oleh VideoLAN. Aplikasi ini dapat memutar berbagai format audio dan video tanpa perlu codec tambahan.

Fitur Utama VLC Media Player:

  • Mendukung Berbagai Format: Memutar hampir semua format audio dan video.
  • Streaming: Mendukung streaming media dari internet.
  • Subtitle: Mendukung berbagai format subtitle dan sinkronisasi.
  • Ekstensi dan Skins: Mendukung ekstensi dan kustomisasi tampilan.

Soal Latihan

  1. Sebutkan dua fitur utama dari VLC Media Player!
  2. Bagaimana cara menambahkan subtitle ke video yang diputar di VLC?
  3. Jelaskan langkah-langkah untuk melakukan streaming media menggunakan VLC Media Player!

Cara Penggunaan VLC Media Player

  1. Buka aplikasi VLC Media Player dari menu Start atau shortcut di desktop.
  2. Klik "Media" di bagian atas, lalu pilih "Open File" untuk memilih file media yang ingin diputar.
  3. Setelah file terbuka, Anda dapat mengontrol pemutaran menggunakan tombol play, pause, dan stop di bagian bawah.
  4. Jika ingin menambahkan subtitle, klik "Subtitle" pada menu atas, lalu pilih "Add Subtitle File".
  5. Untuk streaming, klik "Media" > "Open Network Stream" dan masukkan URL streaming yang diinginkan.
