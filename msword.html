Microsoft Word

Apa Itu Microsoft Word?

Microsoft Word adalah aplikasi pengolah kata yang dikembangkan oleh Microsoft. Aplikasi ini digunakan untuk membuat, mengedit, dan memformat dokumen teks seperti laporan, surat, dan esai.

Fitur Utama Microsoft Word:

  • Pengetikan dan pemformatan teks.
  • Menyisipkan gambar, tabel, dan grafik.
  • Penggunaan template dokumen untuk mempermudah pembuatan dokumen standar.
  • Fitur kolaborasi dan komentar untuk bekerja sama dalam satu dokumen.

Soal Latihan

  1. Sebutkan tiga fitur utama dari Microsoft Word!
  2. Bagaimana cara menyisipkan gambar di Microsoft Word?
  3. Jelaskan langkah-langkah untuk menyimpan dokumen sebagai PDF di Microsoft Word!

Cara Penggunaan Microsoft Word

  1. Buka aplikasi Microsoft Word dari menu Start atau shortcut di desktop.
  2. Pilih template atau dokumen kosong untuk memulai dokumen baru.
  3. Ketik teks yang ingin Anda masukkan. Gunakan toolbar di atas untuk memformat teks sesuai kebutuhan (misalnya mengubah font, ukuran, atau warna).
  4. Untuk menyisipkan gambar, klik tab "Insert", lalu pilih "Pictures" dan cari gambar yang ingin ditambahkan.
  5. Setelah selesai, simpan dokumen dengan mengklik "File" > "Save As" dan pilih lokasi serta format yang diinginkan.
